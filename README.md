# ⚡ QSH

QSH — минималистичная CLI-утилита на Python для быстрых запросов к нейросети прямо из терминала.

Задача: получать короткие, полезные ответы, не отвлекаясь от работы.

---

🚀 Возможности

- Быстрые запросы к LLM прямо из терминала
- Минималистичный интерфейс
- Поддержка Gemini API
- Установка как системной команды (qsh)

---

📦 Установка

git clone https://github.com/ivanzhugg/Qsh
cd qsh

pip install -e .

После этого команда qsh станет доступна в терминале.

---

🔑 Настройка

Укажите API-ключ Gemini в файле:

qsh/data/.env

Пример:

GEMINI_API_KEY=your_api_key_here

---

💻 Использование

qsh как\ проверить\ ip\ адрес

или

qsh "как проверить ip адрес"

---

🧠 О проекте

QSH создан для тех, кто не хочет тратить время на:
- открытие браузера
- переключение вкладок
- лишние действия

Ты просто задаёшь вопрос — и сразу получаешь ответ в терминале.
